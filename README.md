# からあげアゲイン

## 機能
- からあげレシピ投稿機能
　- Advance Settingモードでより詳細なからあげレシピを記録可能
- からあげレビュー投稿機能
- からあげ摂取記録機能
- 今日のからあげメッセージボイス再生機能
- SNS系機能（予定）
- からあゲーム（予定）

## 技術スタック
- Docker
- React
- Tailwind CSS
- Supabase
  - 認証、データベース、画像投稿用ストレージ
- Versel
